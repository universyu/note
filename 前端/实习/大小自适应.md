# 大小自适应

### Paper跟随Dialog变化大小

把Dialog设置为根元素，监听其大小的变化，并随之更改Paper的大小

```tsx
    <Dialog
      open={previewRodinOpen}
      ref={rootRef}
      PaperProps={{
        style: {
          maxWidth: 'none',
          width: 800,
          height: paperHeight,
          display: 'flex',
          flexDirection: 'row',
          borderRadius: 16,
          boxSizing: 'border-box',
        },
      }}
    >
```

下面是监听代码，确保Paper高度在300~600

```tsx
  const [paperHeight, setPaperHeight] = useState(600)
  useLayoutEffect(() => {
    if (rootRef.current) {
      const resizeObserver = new ResizeObserver((entries) => {
        const rootElement = rootRef.current
        if (!rootElement) {
          return
        }
        setPaperHeight(Math.max(300, Math.min(rootElement.clientHeight, 700)) - 100)
      })
      resizeObserver.observe(rootRef.current)
      return () => {
        resizeObserver.disconnect()
      }
    }
  }, [rootRef.current])
```



### canvas占满Paper空闲空间

其他部分非必要不写高度，只写margin

```tsx
  const canvasHeight = useMemo(() => {
    return paperHeight - 166
  }, [paperHeight])
```



用高度为24px的div装关闭按钮，占用24px

```tsx
        <div
          style={{
            height: 24,
            width: '100%',
            margin: 0,
          }}
        >
          <Close
            style={{
              position: 'absolute',
              right: 18,
              top: 12,
              cursor: 'pointer',
            }}
            onClick={() => {...0}}
          />
        </div>
```

高20px的p标签，占用20+30=50px

```tsx
        <p
          style={{
            margin: 0,
            marginBottom: 30,
            padding: 0,
            fontSize: 18,
            fontWeight: 600,
            lineHeight: '20px',
          }}
        >
          ...
        </p>
```

canvas

### 左导航栏

让文字固定在上面，下面的部分可以滑动，为了防止子元素的部分超出父元素而看不见，需要flex-start，默认继承父元素Paper的高度

```tsx
      <CenterPositionDiv
        style={{
          width: 200,
          flexDirection: 'column',
          justifyContent: 'flex-start',
          backgroundColor: '#f7f7f7',
          borderRight: '1px solid #ebebeb',
          boxSizing: 'border-box',
        }}
      >
        <p
          style={{
            margin: '20px 0',
            padding: 0,
            fontSize: 14,
            lineHeight: '20px',
          }}
        >
          ...
        </p>
        <CenterPositionDiv
          style={{
            overflowY: 'scroll',
            scrollbarWidth: 'none',
            justifyContent: 'flex-start',
            flexDirection: 'column',
          }}
        >
          {rodinImages.map((image, index) => (
            <img
              onClick={...}
              style={{
                border: userPickModelIndex === index ? '2px solid #00ae42' : '2px solid #ebebeb',
                borderRadius: 8,
                marginBottom: 20,
                cursor: 'pointer',
              }}
              src={image}
              alt={`model-${index}`}
            />
          ))}
        </CenterPositionDiv>
      </CenterPositionDiv>
```

Tips：这里的CenterPositionDiv如下

```tsx
const CenterPositionDiv = styled('div')(({ theme }) => ({
  display: 'flex',
  justifyContent: 'center',
  alignItems: 'center',
}))
```

### 右区域

只需要一个宽600px的div

```tsx
      <CenterPositionDiv
        style={{
          width: 600,
          flexDirection: 'column',
        }}
      >
```

现在的Paper写宽度800px，左区域写宽度200px，右区域写宽度600px，当宽度不够时会自动等比缩小
